# 🧠 MannMitra - AI Mental Health Support Chatbot

## 📌 Project Description

MannMitra is an AI-based mental health support chatbot designed to provide emotional assistance to users.
It understands user input (like stress, anxiety, or sadness) and responds with supportive, empathetic, and helpful suggestions.

---

## 🚀 Features

* Detects user emotions (anxiety, stress, etc.)
* Provides empathetic responses
* Suggests coping techniques (breathing, task planning, etc.)
* Interactive chatbot interface

---

## 🛠️ Tech Stack

* Python
* Flask
* HTML/CSS (Frontend UI)
* JavaScript (optional for interaction)

---

## ▶️ How to Run the Project

### Step 1: Clone Repository

```bash
git clone https://github.com/HarjotKaur-15/MannMitra-AI-Chatbot.git
cd MannMitra-AI-Chatbot
```

### Step 2: Install Requirements

```bash
pip install -r requirements.txt
```

### Step 3: Run the App

```bash
python app.py
```

### Step 4: Open in Browser

Go to:

```
http://127.0.0.1:5000/
```
